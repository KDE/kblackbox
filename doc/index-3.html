3. GUI description

  • Menus
    • File
      • Quit - Well, it's difficult to explain. Let's skip it.
    • Game
      • New - Starts a new game (and abandons the current, if any).
      • Give up - Shows you positions of the balls.
      • Done - Checks whether all balls are placed. If yes, it terminates the current game, computes the final score and indicates real positions of the balls. The middle mouse button has the same function.
      • Resize - Resizes the main window, so that it fits perfectly its contents. This is useful, when you accidentaly change the size of the window...
    • Options
      • Size - Sets the size of the game field (black box).
      • Balls - Sets the number of balls in the black box.
      • Tutorial - Switches on/off the tutorial mode (i.e you can see, where the balls actually are).
    • Help
      • Help - Displays this help.
      • About - Displays the "About" dialog.
      • About Qt - Displays the information dialog about the Qt GUI library.
  • Tool bar: The tool bar offers some of the most useful menu functions, namely, in the mentioned order:
    • Quit
    • New
    • Give up
    • Done
  • Status bar: It shows you some useful information.
